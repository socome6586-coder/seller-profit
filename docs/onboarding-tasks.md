# 작업지시서 — 계정 연동 가이드 + 대시보드 온보딩 개선 (Claude Code 용)

> 저장소 위치 권장: `docs/onboarding-tasks.md`
> 배경: 실제 운영 사이트(`sellerprofit.co.kr`)의 "계정 연동" 화면을 확인한 결과, 업체코드/
> Access Key/Secret Key 입력칸만 있고 **이걸 어디서 구하는지 안내가 전혀 없다.** 쿠팡 API를
> 처음 다뤄보는 셀러는 이 화면에서 막힌다 — 기능이 아무리 좋아도 여기서 이탈하면 소용없다.
> 기존 규약 계승: 프론트 수정 후 `npm run build` 산출물 커밋, 페이퍼/잉크 디자인 톤 유지
> (`styles.css` 기존 토큰 재사용, 새 색상 도입 금지).

## ⚠️ 절대 규칙

- 가이드 문구는 **실제 쿠팡 WING 화면 기준으로 정확하게** 쓴다(추측 금지). 아래 §1에 검증된
  절차를 그대로 반영.
- **서버 공인 IP를 코드 여러 곳에 하드코딩하지 않는다.** 한 곳(설정값 또는 상수)에 정의해서
  가이드 화면이 그 값을 참조하게 한다 — 서버 이전 시 한 곳만 고치면 되도록.

## 1. 계정 연동 가이드에 반영할 실제 절차 (검증됨)

1. 쿠팡 **WING**(wing.coupang.com) 로그인
2. 우측 상단 아이디 클릭 → **"판매자정보"**(또는 **"추가판매정보"**) 클릭
3. 페이지 하단 **"OPEN API 키 발급받기"** 버튼 클릭
4. 팝업에서 **"OPEN API"** 선택 → 약관 동의 → **"연동업체 선택"** 화면에서
   **"자체개발(직접입력)"** 선택(seller-profit이 목록에 없으므로) → 아래 정보 입력:
   - 업체명: 아무 값(예: 개인 상호명)
   - URL: 없으면 `wing.coupang.com` 입력해도 무방
   - **IP 주소: ⚠️ 반드시 seller-profit 운영 서버의 공인 IP를 입력**(§2 참고).
     본인 PC IP 가 아님에 주의 — 실제 API 호출은 seller-profit 서버에서 나간다.
5. 발급 완료 화면에서 **업체코드 / Access Key / Secret Key** 확인 후 복사
6. seller-profit "계정 연동" 화면에 그대로 붙여넣기

- 참고: 사업자 인증이 안 된 일반회원은 API Key 발급 자체가 안 된다(이미 활성 쿠팡 셀러는
  통과된 상태이므로 보통 문제 없음). API 재발급/정보수정은 월 10회 제한, 유효기간 180일.

## 2. 서버 공인 IP 관리 원칙

- 현재 값: `49.247.139.234`(iwinv 서버). **이 값을 프론트 코드에 문자열로 흩뿌리지 말고**
  한 곳에서 관리한다 — 예: 백엔드 설정값(`application.yml` 의 `app.public-server-ip` 등)을
  두고 프론트가 API로 받아 표시하거나, 최소한 프론트 내 단일 상수(`SERVER_PUBLIC_IP`)로
  정의해 가이드 컴포넌트가 그 값만 참조하게 한다.
- 서버 IP가 변경되면 이 한 곳만 고치면 전체 가이드 문구가 갱신되도록 구조를 짠다.
- **AC:** 코드에서 `49.247.139.234` 문자열이 2곳 이상 중복되지 않는다(grep 으로 확인).

## T13.2 — 대시보드 빈 상태(온보딩) 개선

- `frontend/src/pages/Dashboard.jsx` 에서 "연결된 마켓 계정이 없습니다" 상태를 단순 경고
  문구에서 **온보딩 유도 카드**로 개선:
  - 문구 예시(참고, 그대로 쓸 필요는 없음): "계정을 연동하면 여기서 적자 상품을
    바로 확인할 수 있어요."
  - **"계정 연동하러 가기" 버튼**을 눈에 띄게 배치 → 클릭 시 `/accounts` 로 이동.
  - 기존 "총매출/총순이익/평균 마진율/배분 기타비용/광고비" 카드들이 전부 `-` 로 뜨는
    현재 모습은, 계정 연동 전에는 아예 안 보이거나 흐리게(disabled 톤) 처리해 "고장난
    화면처럼" 보이지 않게 한다.
  - 계정 연동 후에는 기존 동작(조회 버튼·프리셋 칩) 그대로 유지 — 이 개선은 **연동 전
    상태에만** 적용.
- **AC:**
